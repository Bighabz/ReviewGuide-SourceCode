# ReviewGuide.ai — Ideas & Future Features

Ideas captured during project planning. Not in current scope — revisit for future milestones.

## Price Intelligence
- **Price alerts**: Users set price targets, get notified (email/push) when product drops below threshold
- **Price tracking**: 30-day / 90-day price history per product (like Rufus)
- **Cross-retailer price comparison**: Same product, different stores, show savings
- **Auto-buy agent**: Set a price target, agent purchases automatically when hit (requires accounts + payment)

## Personalization & Accounts
- **Optional user accounts**: Save history, preferences, wishlists — works without login
- **Required accounts (later)**: Full personalization, recommendations based on past behavior
- **Shopping profile**: Budget preferences, brand affinities, size/fit preferences
- **Conversation memory**: Remember past searches and preferences across sessions

## Affiliate Network Expansion
- **Impact.com**: REST API, AI/blockchain tracking, Shopify integration
- **Rakuten Advertising**: Direct advertiser relationships
- **Awin**: Publisher API with link builder, 20 calls/min rate limit
- **Skimlinks** (priority): One integration → 48,500 merchants, 50+ networks. Use as catch-all aggregator, keep direct relationships (Amazon, Booking, Viator) for better rates
- **Booking.com**: Direct affiliate program for hotels
- **More CJ advertisers**: Best Buy, Dell, HP, Samsung, Target, Wayfair, Nike, Nordstrom

## Verticals to Add
- **Automobile shopping & rentals**: Car comparisons, dealer pricing, rental aggregation
- **Car parts**: Compatibility matching, OEM vs aftermarket
- **Mattresses**: Sleep profile matching, firmness recommendations
- **Home goods & furniture**: Room-based recommendations
- **Fashion & beauty**: Style matching, size recommendations
- **Excursions**: Viator-style activity booking with itinerary integration

## Rufus-Inspired Features
- **"Help Me Decide" mode**: Guided narrowing when user is overwhelmed by choices
- **Use-case discovery**: "What do I need for cold weather golf?" → full kit recommendation
- **Post-purchase support**: Order tracking, return guidance (requires retailer API access)
- **Handwritten list parsing**: Upload grocery/shopping list → find all items
- **Conversational refinement**: Multi-turn narrowing ("now show me only under $200 with good reviews")

## Technical Ideas
- **Review aggregation scoring**: Weighted sentiment across multiple review sources
- **Product knowledge graph**: Build relationships between products, categories, use-cases
- **Affiliate link health monitoring**: Auto-detect broken links, swap to working alternatives
- **A/B test response formats**: Blog-style vs card-style vs hybrid — track conversion
- **SEO landing pages**: Auto-generate category pages from popular queries
