- [ ] Open https://www.reviewguide.ai in Chrome desktop
- [ ] Open https://www.reviewguide.ai on phone (or Chrome DevTools 375x812)

## 2. Chat Page (/chat)

### Message Sending
- [ ] Type a message and send → message appears as blue bubble on right
- [ ] **CRITICAL: sent message stays visible on mobile** (not scrolled off screen)
- [ ] Loading dots appear in AI response bubble
- [ ] Blog text streams in real-time (words appear progressively)

### Product Query Tests
| Query | Expected | Check |
|-------|----------|-------|
| "best electric skateboards" | Product cards + blog article | [ ] |

### Product Cards (ProductReview)
- [ ] Product name is bold, readable
- [ ] Product image loads (not broken)
- [ ] Pros section uses green text (var(--success))
- [ ] Cons section uses red text (var(--error))
- [ ] "Where to Buy" section shows max 3 offers
- [ ] Merchant names are clean ("eBay" not "eBay (lawrenow-0)")
- [ ] Prices show "$249.99" not "USD 249.99"
- [ ] Amazon links → amazon.com (not ebay.com)
- [ ] "Check price →" links → amazon.com/s?k=... with tag=revguide-20

### Product Cards — Mobile
- [ ] Cards have tighter padding (12px vs 24px)
- [ ] Product images are 64px (not 112px)
- [ ] "Where to Buy" links stack vertically (single column)
- [ ] No horizontal overflow / distortion

### TopPickBlock
- [ ] "Our Top Pick" badge visible
- [ ] Product image loads
- [ ] "Buy on [merchant]" button visible and clickable
- [ ] Button text matches actual URL domain (Amazon → "Buy on Amazon")
- [ ] **Mobile: image stacks above text** (not side-by-side)
- [ ] **Mobile: CTA button is not cut off**

### Sources
- [ ] Source links (RTINGS, Tom's Guide, CNET etc.) are clickable
- [ ] Links open in new tab
- [ ] Colored dots appear next to source names
