  <action>
    Create `frontend/components/ResultsProductCard.tsx` as a `'use client'` component.

    **Props interface:**
    ```typescript
    interface ResultsProductCardProps {
      product: ExtractedProduct  // from extractResultsData.ts
      index: number              // position in list (0-based), used for rank + score + accent color
    }
    ```

    Import ExtractedProduct from `@/lib/extractResultsData`.

    **Copy lookupCuratedProduct function from InlineProductCard.tsx** — identical logic for curated Amazon image and affiliate URL lookup. Import curatedLinks from `@/lib/curatedLinks`.

    **Card structure per CONTEXT.md locked decisions:**

    Outer wrapper: `rounded-2xl` (16px), `border` with `borderColor: var(--border)`, `background: var(--surface-elevated)`, `padding: 12px` (p-3), `overflow-hidden`. Add the `product-card-hover` class for lift effect on desktop.

    **Top section (100px height):**
    - Background: `var(--card-accent-${(index % 4) + 1})` — deterministic, no Math.random()
    - `rounded-xl` inner container, `h-[100px]`, `relative`, `flex items-center justify-center`
    - Rank badge: absolute top-left (top-2 left-2), `w-6 h-6` (24px) black circle, white text number (`index + 1`), `rounded-full bg-black text-white text-xs font-bold flex items-center justify-center`
    - Product image: `w-16 h-16` (~64px), `object-contain`. Use lookupCuratedProduct for curated image, fall back to product.image_url, then to a category placeholder icon (ShoppingCart from lucide-react). Use useState for image error state (same pattern as InlineProductCard.ProductImage).

    **Middle section:**
    - Category badge pill: Determine from index — index 0: "Top Pick" with warm gold bg (`#B8860B` text on `rgba(184,134,11,0.1)` bg), index 1: "Best Value" with blue (`var(--primary)` text on `var(--primary-light)` bg), index 2: "Premium" with purple (`#7C3AED` text on `rgba(124,58,237,0.1)` bg), index >= 3: `#${index+1}` with neutral styling. Pill: `px-2 py-0.5 rounded-full text-[11px] font-medium inline-block mb-1`.
    - Product name: `font-semibold text-sm` (14px SemiBold), `line-clamp-2`, `style={{ color: 'var(--text)' }}`.
    - Description: `text-[11px]` (11px Regular), `line-clamp-2`, `style={{ color: 'var(--text-secondary)' }}`. Show product.description if available.
    - Score bar: Derive score from position — `const POSITION_SCORES = [95, 88, 82, 76, 70]` with fallback to 60. Score bar is `h-1` (4px), `rounded-full`, background `var(--surface-hover)`, inner fill `var(--primary)`, width = `${score}%`. Score number right-aligned `text-xs font-medium` in `var(--text-secondary)`. Use `role="progressbar"` and `aria-valuenow={score}` for accessibility and testability.

    **Bottom section:**
    - Price: `font-bold text-lg` (18px Bold), `style={{ color: 'var(--text)' }}`. Display as `$${product.price}` if price exists, otherwise hide.
    - CTA button: If affiliate URL exists (from curated lookup or product.url): `<a>` tag styled as button — `var(--primary)` background, white text, `rounded-lg` (8px radius), `h-9` (36px height), `px-4`, `text-sm font-medium`, text "Buy on Amazon", `target="_blank" rel="noopener noreferrer"`. If no affiliate URL: muted text link "Check Amazon →" in `var(--text-muted)`, linking to Amazon search `https://www.amazon.com/s?k=${encodeURIComponent(name)}&tag=revguide-20`.
    - Layout: `flex items-center justify-between` for price-left, CTA-right.

    **Width behavior:**
    - The card itself has NO fixed width — the parent container controls width via CSS grid (desktop) or flex with snap (mobile). The card uses `w-full` internally.
    - On mobile: parent will set `min-w-[170px] max-w-[170px]` via snap container
    - On desktop: parent CSS grid `grid-cols-3` auto-sizes

    Use CSS variables exclusively (var(--*)) for all colors. No Tailwind dark: utilities.
