  <name>Task 1: Create EditorsPicks component with product images</name>
  <files>frontend/components/EditorsPicks.tsx</files>
  <action>
Create `frontend/components/EditorsPicks.tsx` as a client component that renders curated products with Amazon CDN images.

**Component structure:**

```
EditorsPicks
  props: { categorySlug: string }
  returns: null if curatedLinks[categorySlug] is undefined or empty
  otherwise: section with heading + topic groups, each with title + product image grid
```

**Implementation details:**

1. Add `'use client'` directive at top (needed for useState in image error handling).

2. Import `{ useState }` from React, `{ ExternalLink, Package }` from lucide-react, `{ curatedLinks }` from `@/lib/curatedLinks`.

3. Create `getImageUrl(asin: string): string` helper that returns `https://images-na.ssl-images-amazon.com/images/I/${asin}._SL300_.jpg` -- the proven pattern from InlineProductCard.tsx line 38.

4. Create a `ProductImage` internal component (same pattern as InlineProductCard.tsx) that:
   - Takes `{ asin, alt }` props
   - Uses `useState(false)` for error state
   - On error: renders a placeholder div with `Package` icon (NOT placehold.co)
   - On success: renders `<img>` with `src={getImageUrl(asin)}`, `alt={alt}`, `loading="lazy"`, and `onError` handler
   - Uses `aspect-square object-cover rounded-lg` classes

5. Create the `EditorsPicks` component:
   - Look up `curatedLinks[categorySlug]`. If falsy or empty array, return `null`.
   - Render a `<section>` with consistent padding: `px-4 sm:px-6 md:px-8 py-8`.
   - Render heading row with: `<h2 className="font-serif text-xl text-[var(--text)] tracking-tight">Editor's Picks</h2>` with the same divider pattern as "Popular Questions" heading (flex items-center gap-3, then h-px bg-[var(--border)] divider).
   - For each topic in the array, render:
     - `<h3>` with topic.title in `font-serif text-base font-semibold text-[var(--text)]`
     - `<p>` with topic.description in `text-sm text-[var(--text-secondary)]`
     - A grid of product cards: `grid grid-cols-2 sm:grid-cols-3 lg:grid-cols-5 gap-3`
     - Each product card is an `<a>` wrapping: ProductImage + rank label ("Option 1", "Option 2", etc.)
   - Each product `<a>` tag:
     - `href={product.url}` (the amzn.to affiliate link from curatedLinks)
     - `target="_blank" rel="noopener noreferrer"`
     - Has a card style: `rounded-xl border border-[var(--border)] bg-[var(--surface-elevated)] overflow-hidden shadow-card product-card-hover`
     - Contains the ProductImage component
     - Below the image: a label area with `text-xs text-[var(--text-secondary)] px-2 py-1.5` showing `Option ${index + 1}`
     - An ExternalLink icon (size 10, opacity-60) next to the label

**Key constraints:**
- Do NOT use `Math.random()` or `Date.now()` anywhere (hydration safety per CLAUDE.md).
- Do NOT use placehold.co -- image fallback must be a styled div with Package icon.
- Do NOT use Tailwind `dark:` utilities -- CSS variables handle dark mode.
- Do NOT modify curatedLinks.ts (read-only data source).
