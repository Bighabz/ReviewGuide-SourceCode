Create `backend/app/services/affiliate/providers/impact_provider.py` implementing the full provider. Use the research code skeleton as the base (lines 187-439 of 10-RESEARCH.md) with these specific requirements:

1. **Imports**: hashlib, json, time, typing (List, Optional), httpx, get_logger, settings, get_redis, redis_get_with_retry, redis_set_with_retry, BaseAffiliateProvider, AffiliateProduct, AffiliateProviderRegistry.

2. **Registration decorator**:
```python
@AffiliateProviderRegistry.register(
    "impact",
    required_env_vars=["IMPACT_ACCOUNT_SID", "IMPACT_AUTH_TOKEN"],
)
```

3. **Constructor** `__init__(self, account_sid=None, auth_token=None)`:
   - Read all settings from `settings.IMPACT_*` with parameter overrides
   - Build `self.search_url = f"{IMPACT_API_BASE}/Mediapartners/{self.account_sid}/Catalogs/ItemSearch"`
   - Use base URL `https://api.impact.com` (NOT Advertisers -- MUST use Mediapartners per research Pitfall 2)
   - Accept `api_enabled` parameter for test override: `api_enabled=None` defaults to `settings.IMPACT_API_ENABLED`, but if explicitly passed True/False, use that value. This is needed for TestImpactFeatureFlag tests.

4. **get_provider_name()**: Return `"Impact.com"`

5. **_check_rate_limit()**: Redis sorted-set sliding window. Key: `"impact:api:rate_limit"`. Window: 3600 seconds. Max: `self.max_requests_per_hour`. On Redis failure, fail OPEN (return True, log warning). Use unique member key `f"{now}:{id(self)}:{time.time_ns()}"` to avoid collisions in concurrent calls.

6. **_build_cache_key(params)**: MD5 hash of sorted JSON params, prefix `"impact:search:"`, truncate hash to 12 chars. Same pattern as CJ.

7. **search_products(query, category, brand, min_price, max_price, limit)**:
   - Early return `[]` if `self.api_enabled` is False
   - Build params: `{"Keyword": query, "PageSize": min(limit, self.max_results)}`
   - If `brand` is provided, add `"Manufacturer": brand`
   - Check Redis cache first (cache hit returns deserialized products)
   - Check rate limit (rate limit exceeded returns `[]`)
   - Call Impact.com API via `httpx.AsyncClient.get` with HTTP Basic Auth `auth=(self.account_sid, self.auth_token)` and `headers={"Accept": "application/json"}`
   - Handle 429 (rate limited by API) -> return `[]`
   - Handle non-200 -> log error, return `[]`
   - Parse response via `_parse_response(response.json())`
   - Cache results via `redis_set_with_retry` with `self.cache_ttl`
   - Catch `httpx.TimeoutException` -> return `[]`
   - Catch general `Exception` -> log with exc_info, return `[]`

8. **_parse_response(data)**: Parse `data["Items"]` list. For each item:
   - `product_id` = `str(item.get("CatalogItemId", ""))`
   - `title` = `item.get("Name", "")`
   - `price` = `float(item.get("CurrentPrice", 0))` (handle both int and float)
   - `currency` = `item.get("Currency", "USD")`
   - `affiliate_link` = `item.get("Url", "")` (pre-built tracking URL, do NOT construct manually)
   - `merchant` = `item.get("CampaignName", "Impact.com")`
   - `image_url` = `item.get("ImageUrl")`
   - `rating` = None (Impact.com does not provide ratings)
   - `review_count` = None
   - `condition` = map from Condition field: New->new, Used->used, Refurbished->refurbished, OEM->new, OpenBox->used, default->new
   - `shipping_cost` = `float(item.get("ShippingRate"))` if present, else None
   - `availability` = StockAvailability in ("InStock", "PreOrder", "LimitedAvailability", "BackOrder")
   - `source_url` = same as affiliate_link
   - Wrap each item parse in try/except, log warning and continue on failure

9. **generate_affiliate_link()**: Return empty string. Impact.com URLs are pre-tracked in the Url field from search results.

10. **check_link_health(affiliate_link)**: HEAD request with 5s timeout, accept 200/301/302 as healthy. Catch exceptions, return False.

11. **Module exports**: `__all__ = ["ImpactAffiliateProvider"]`

Critical: The `api_enabled` constructor parameter must support being passed explicitly (for tests) or falling back to settings (for production). Implementation:

```python
def __init__(self, account_sid=None, auth_token=None, api_enabled=None):
    ...

```
