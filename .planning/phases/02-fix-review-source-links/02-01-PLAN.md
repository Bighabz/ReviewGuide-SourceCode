Review sources (Wirecutter, Tom's Guide, Reddit) exist in the pipeline's review_bundles data
but are never passed to the LLM or surfaced to the user.

Purpose: Trust and transparency. Editorial citations are the core differentiator — AI shopping
recommendations without source links look fabricated.
Output: Modified product_compose.py with source URLs threaded into LLM context, system prompt
updated to instruct citation linking, and citations array populated from review sources.

```python
# Thread review source URLs into LLM context
source_refs = ""
if bundle.get("sources"):
    top_sources = bundle["sources"][:3]
    source_parts = [
        f"[{s.get('site_name', 'source')}]({s.get('url', '')})"
        for s in top_sources
        if s.get("url")

    if source_parts:
        source_refs = " | Reviews: " + ", ".join(source_parts)
```
