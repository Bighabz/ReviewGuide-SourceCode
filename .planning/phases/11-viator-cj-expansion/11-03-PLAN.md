```python
TOOL_CONTRACT = {
    "name": "travel_search_activities",
    "purpose": "Search for tours, activities, experiences, things to do",
    "intent": "travel",
    "tools": {"pre": [], "post": []},
    "produces": ["activities"],
    "required_slots": ["destination"],
    "optional_slots": ["departure_date", "duration_days"],
    "citation_message": "Finding activities and experiences...",
    "tool_order": 100,
    "slot_types": {
        "destination": {"type": "string", "format": "city"},
        "departure_date": {"type": "date", "format": "YYYY-MM-DD"},
        "duration_days": {"type": "integer", "format": "number"},
    }
}
```

2. **`travel_search_activities(state)` function:**
   - Read `destination` from `state.get("slots", {}).get("destination", "")`
   - Instantiate `ViatorActivityProvider` with `api_key=settings.VIATOR_API_KEY`, `affiliate_id=settings.VIATOR_AFFILIATE_ID`
   - Call `await provider.search_products(query=destination, limit=settings.VIATOR_MAX_RESULTS)`
   - Convert each `AffiliateProduct` to a dict for state:
     ```python
     activity_dict = {
         "type": "activity",
         "provider": "viator",
         "name": product.title,
         "price_from": product.price,
         "currency": product.currency,
         "booking_url": product.affiliate_link,
         "image_url": product.image_url,
         "rating": product.rating,
         "review_count": product.review_count,
         "merchant": product.merchant,
     }
     ```
   - Return `{"activities": activities_list, "citations": [link for each activity], "success": len(activities_list) > 0}`
   - Wrap in `@tool_error_handler` decorator
   - On exception: return `{"activities": [], "citations": [], "error": str(e), "success": False}`
