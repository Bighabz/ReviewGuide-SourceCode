**File 1: `backend/app/services/affiliate/providers/viator_provider.py`**

Create following the CJ provider pattern exactly:

1. **Imports:** `hashlib`, `json`, `httpx`, `List`, `Optional`, `get_logger`, `settings`, `redis_get_with_retry`, `redis_set_with_retry`, `BaseAffiliateProvider`, `AffiliateProduct`, `AffiliateProviderRegistry`, `ViatorPLPLinkGenerator`.

2. **Registry decorator:**
```python
@AffiliateProviderRegistry.register(
    "viator",
    required_env_vars=["VIATOR_API_KEY"],
    optional_env_vars=["VIATOR_AFFILIATE_ID"],
)
class ViatorActivityProvider(BaseAffiliateProvider):
```

3. **Constants:**
```python
VIATOR_API_BASE = "https://api.viator.com/partner"
```

4. **`__init__`:** Accept `api_key` and `affiliate_id` kwargs with fallback to settings. Store `self.api_key`, `self.affiliate_id`, `self.timeout` (from settings.VIATOR_API_TIMEOUT), `self.cache_ttl` (from settings.VIATOR_CACHE_TTL), `self.max_results` (from settings.VIATOR_MAX_RESULTS).

5. **`get_provider_name()`:** Return `"Viator"`.

6. **`_build_cache_key(params)`:** Same pattern as CJ -- MD5 hash of sorted JSON params, prefix `viator:search:`.

7. **`_build_affiliate_link(product_url)`:** Append `?pid={self.affiliate_id}&mcid=42383&medium=api` to the product URL. Use `&` if `?` already in URL.

8. **`search_products(query, category, brand, min_price, max_price, limit)`:**
   - Build Redis cache key from query + limit
   - Check cache with `redis_get_with_retry`; on hit, deserialize and return `List[AffiliateProduct]`
   - On cache miss, call Viator API:
     ```python
     headers = {
         "exp-api-key": self.api_key,
         "Accept-Language": "en-US",
         "Accept": "application/json;version=2.0",
         "Content-Type": "application/json",
     }
     body = {
         "searchTerm": query,
         "searchTypes": [{"searchType": "PRODUCTS", "pagination": {"start": 1, "count": min(limit, self.max_results)}}],
         "currency": "USD",
         "productSorting": {"sort": "DEFAULT", "order": "DESCENDING"},
     }
     ```
   - POST to `{VIATOR_API_BASE}/search/freetext`
   - Parse response with `_parse_api_response`
   - Cache serialized products with `redis_set_with_retry` (TTL = self.cache_ttl)
   - **On any exception** (httpx.TimeoutException, httpx.ConnectError, other): Log error, fall back to PLP link:
     ```python
     fallback_url = ViatorPLPLinkGenerator.generate_activity_search_url(query)
     return [AffiliateProduct(
         product_id="viator-plp-fallback",
         title=f"Activities in {query}",
         price=0.0,
         currency="USD",
         affiliate_link=fallback_url,
         merchant="Viator",
         image_url=None,
         source_url=fallback_url,
     )]
     ```
   - Return empty list `[]` on API error status codes (non-200).

9. **`_parse_api_response(data: dict)`:** Extract from response JSON:
   - Navigate `data["products"]["results"]` (list of product dicts)
   - For each product:
     - `product_id`: `f"viator-{product.get('productCode', '')}"`
     - `title`: `product.get("title", "Unknown Activity")`
     - `price`: `product.get("pricing", {}).get("summary", {}).get("fromPrice", 0.0)`
     - `currency`: `product.get("currency", "USD")`
     - `affiliate_link`: `self._build_affiliate_link(product.get("productUrl", ""))`
     - `image_url`: `product.get("thumbnailUrl", "")`
     - `rating`: `product.get("rating", None)` (float)
     - `review_count`: `product.get("reviewCount", None)` (int)
     - `merchant`: `"Viator"`
   - Return `List[AffiliateProduct]`

10. **`generate_affiliate_link(product_id, campaign_id, tracking_id)`:** Build Viator deep link using product_id. Return `f"https://www.viator.com/tours/{product_id}?pid={self.affiliate_id}&mcid=42383&medium=api"`.

11. **`check_link_health(affiliate_link)`:** Same as CJ -- HEAD request, accept 200/301/302, catch exceptions and return False.
