    **Fix 1 — Fallback loop (line ~1063-1066):**
    Change the loop from:
    ```python
    for pname in blog_product_names:
        if pname in seen_card_names or review_card_count + fallback_card_count >= 5:
            break
    ```
    To:
    ```python
    for pname in blog_product_names:
        if review_card_count + fallback_card_count >= 5:
            break   # cap reached — stop entirely
        if pname in seen_card_names:
            continue   # skip duplicate but keep iterating
    ```

    **Fix 2 — Multi-provider gating (line ~937-944):**
    Change from requiring `len(providers_in_offers) >= 2` with narrow curated/eBay exceptions to simply requiring `len(real_offers) >= 1` (at least one real offer with a valid URL). Keep the provider set computation for ranking purposes but remove it as a hard gate.

    **Fix 3 — Label-domain parity:**
    After constructing `where_to_buy` offer objects, add a validation pass: for each offer, extract the domain from `offer["url"]` and verify it matches the `offer["source"]` label. Add a helper function `_domain_to_merchant(url: str) -> str` that maps common domains to merchant names.

    **Fix 4 — Citation URL validation (line ~1233):**
    Add a filter: `review_source_urls = [u for u in review_source_urls if u and u.startswith("http")]` before the `[:5]` slice.
