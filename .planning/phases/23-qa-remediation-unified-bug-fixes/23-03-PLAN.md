Streaming status pattern (from chat.py lines 459-470):
The SSE handler in chat.py already processes stream_chunk_data with type "tool_citation" as status messages:
```python
if data_type == "tool_citation" and isinstance(data_content, dict):
    citation_message = data_content.get("message", "")
    if citation_message:

```
